> ‚ÄúWe can process the number in chunks of three digits (hundreds). Each chunk represents ‚ÄòBillion‚Äô, ‚ÄòMillion‚Äô, ‚ÄòThousand‚Äô, or the last part. For each chunk, I‚Äôll convert numbers ‚â§999 into words using helper logic, then combine all chunks with their scales.‚Äù

‚úÖ Step-by-step logic:

  1. Special case: if num == 0, return "Zero".
  2. Split num into groups of 3 digits:
  • billions = num / 1,000,000,000
  • millions = (num / 1,000,000) % 1000
  • thousands = (num / 1,000) % 1000
  • remainder = num % 1000
  1. For each non-zero group, convert it to English and append "Billion", "Million", "Thousand", etc.
  2. Trim spaces at the end and capitalize properly.

‚úÖ Java Implementation

class Solution {
    private final String[] BELOW_20 = {"", "One", "Two", "Three", "Four", "Five", "Six",
                                       "Seven", "Eight", "Nine", "Ten", "Eleven", "Twelve",
                                       "Thirteen", "Fourteen", "Fifteen", "Sixteen", "Seventeen",
                                       "Eighteen", "Nineteen"};

    private final String[] TENS = {"", "", "Twenty", "Thirty", "Forty", "Fifty",
                                   "Sixty", "Seventy", "Eighty", "Ninety"};

    private final String[] THOUSANDS = {"", "Thousand", "Million", "Billion"};

    public String numberToWords(int num) {
        if (num == 0) return "Zero";

        int i = 0;
        String words = "";

        while (num > 0) {
            if (num % 1000 != 0) {
                words = helper(num % 1000) + THOUSANDS[i] + " " + words;
            }
            num /= 1000;
            i++;
        }

        return words.trim();
    }

    private String helper(int num) {
        if (num == 0) return "";
        else if (num < 20) return BELOW_20[num] + " ";
        else if (num < 100)
            return TENS[num / 10] + " " + helper(num % 10);
        else
            return BELOW_20[num / 100] + " Hundred " + helper(num % 100);
    }


}
