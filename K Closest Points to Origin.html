K Closest Points to Origin

Given an array of points where points[i] = [xi, yi] represents a point on the X-Y plane and an integer k, return the k closest points to the origin (0, 0).

The distance between two points on the X-Y plane is the Euclidean distance (i.e., ‚àö(x1 - x2)2 + (y1 - y2)2).

You may return the answer in any order. The answer is guaranteed to be unique (except for the order that it is in).

}

}

Example 1:

Input: points = [[1,3],[-2,2]], k = 1 Output: [[-2,2]] Explanation: The distance between (1, 3) and the origin is sqrt(10). The distance between (-2, 2) and the origin is sqrt(8). Since sqrt(8) < sqrt(10), (-2, 2) is closer to the origin. We only want the closest k = 1 points from the origin, so the answer is just [[-2,2]]. Example 2:

Input: points = [[3,3],[5,-1],[-2,4]], k = 2 Output: [[3,3],[-2,4]] Explanation: The answer [[-2,4],[3,3]] would also be accepted.

Constraints:

1 <= k <= points.length <= 10^4 -10^4 <= xi, yi <= 10^4

Approaches (brief)

  • Max-heap (priority queue): push points with key = distance¬≤, keep size ‚â§ k by evicting largest. Final heap contains k closest points. Simpler to implement; time O(n log k), space O(k).
  • Quickselect (chosen): partition points by distance¬≤ and recursively shrink to find k-th smallest distance; then return first k points. Average time O(n), worst-case O(n¬≤) but practical and faster for large inputs.

I‚Äôll show the Quickselect implementation and include a small heap-based alternative.

---

Java ‚Äî Quickselect implementation (average O(n) time, O(1) extra space)

import java.util.Random;

class Solution {
    private final Random rand = new Random();

    public int[][] kClosest(int[][] points, int k) {
        if (k >= points.length) return points.clone();

        // quickselect partition to put k closest points in points[0..k-1]
        quickSelect(points, 0, points.length - 1, k - 1);

        int[][] res = new int[k][2];
        for (int i = 0; i < k; i++) res[i] = points[i];
        return res;
    }

    // quickselect to ensure that indices <= target are the smallest distances
    private void quickSelect(int[][] pts, int left, int right, int targetIdx) {
        while (left <= right) {
            int pivotIndex = left + rand.nextInt(right - left + 1);
            int pivotPos = partition(pts, left, right, pivotIndex);
            if (pivotPos == targetIdx) return;
            else if (pivotPos < targetIdx) left = pivotPos + 1;
            else right = pivotPos - 1;
        }
    }

    // partition around pivotIndex, return final pivot position
    private int partition(int[][] pts, int left, int right, int pivotIndex) {
        long pivotDist = dist2(pts[pivotIndex]);
        swap(pts, pivotIndex, right); // move pivot to end
        int store = left;
        for (int i = left; i < right; i++) {
            if (dist2(pts[i]) <= pivotDist) {
                swap(pts, store, i);
                store++;
            }
        }
        swap(pts, store, right); // move pivot to its final place
        return store;
    }

    private long dist2(int[] p) {
        long x = p[0], y = p[1];
        return x * x + y * y;
    }

    private void swap(int[][] a, int i, int j) {
        int[] tmp = a[i];
        a[i] = a[j];
        a[j] = tmp;
    }

    private static void print(int[][] arr) {
        System.out.print("[");
        for (int i = 0; i < arr.length; i++) {
            System.out.print("[" + arr[i][0] + "," + arr[i][1] + "]");
            if (i + 1 < arr.length) System.out.print(",");
        }
        System.out.println("]");
    }
}

Heap alternative (simpler to reason about)

// Use a max-heap of size k keyed by distance squared.
PriorityQueue pq = new PriorityQueue<>((a,b) -> Long.compare(dist2(b), dist2(a)));
for (int[] p : points) {
    pq.offer(p);
    if (pq.size() > k) pq.poll(); // remove farthest among seen
}
// pq contains k closest points

Complexity

  • Quickselect: average O(n) time, O(1) extra space (in-place). Worst-case O(n¬≤) but random pivot avoids typical worst-cases.
  • Heap: O(n log k) time, O(k) space.
