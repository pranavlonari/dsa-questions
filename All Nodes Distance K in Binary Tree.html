All Nodes Distance K in Binary Tree

Medium
Binary TreeTreeBFSDFSGraph

Problem restatement

Given the root of a binary tree, a target node value, and integer k, return the values of all nodes that are distance k from the target node. Distance is measured by edges. Return nodes in any order.

Input (example 1) root = [3,5,1,6,2,0,8,null,null,7,4], target = 5, k = 2 Output: [7,4,1] (order can vary)

Input (example 2) root = [1], target = 1, k = 3 Output: []

Constraints (provided):

  • number of nodes ‚àà [1, 500]
  • 0 <= Node.val <= 500, all values unique
  • target is guaranteed to be the value of one node in the tree
  • 0 <= k <= 1000

---

Quick script to say to the interviewer

  1. "I'll convert the tree into an undirected graph by recording parent pointers for each node, then run a BFS from the target node to collect nodes at distance k."
  2. "This is O(n) time and O(n) space and is straightforward to implement and reason about."
  3. "Alternatively, there is a recursive DFS-only approach that returns distances from target and collects nodes, which avoids an explicit parent map ‚Äî but BFS+parent is simpler and clear."

---

Approaches considered (pros / cons)

  1. Parent-map + BFS (chosen)
  • Pros: Simple, easy to code, intuitive: BFS naturally gives distance layers.
  • Cons: Extra O(n) space for parent pointers.
  1. Single recursive DFS that returns distances
  • Pros: No explicit parent map; more elegant recursion.
  • Cons: Slightly trickier to implement correctly in interview under time pressure.
  1. Transform to graph and run BFS (equivalent to parent-map approach).

I choose parent-map + BFS for clarity and reliability.

---

Detailed chosen approach (step-by-step)

  • Data structures: Map parent to map child ‚Üí parent; Queue for BFS; Set visited to avoid revisits.
  • Steps:
  1. DFS (or iterative traversal) to fill parent pointers and also find the targetNode matching the given target value.
  2. BFS starting from targetNode, level by level. Maintain a dist counter or iterate level-by-level for k rounds:
  • If k == 0, return [target.val].
  • For each node popped, push its left, right, and parent (if not null and not visited).
  1. When BFS has performed k levels, the nodes in the queue are exactly distance k. Collect their values and return.
  • Correctness: BFS explores all nodes at increasing distances from the target; parent pointers allow moving up the tree.

---

Complexity

  • Time: O(n) ‚Äî one traversal to build parents + BFS up to n nodes.
  • Space: O(n) ‚Äî parent map, visited set and queue may hold up to O(n) nodes.

Java implementation (clean & commented)

import java.util.*;

// Definition for a binary tree node.
class TreeNode {
    int val;
    TreeNode left;
    TreeNode right;
    TreeNode(int x) { val = x; }
}

public class NodesDistanceK {
    public List distanceK(TreeNode root, int targetVal, int k) {
        if (root == null) return Collections.emptyList();

        // 1) Build parent pointers and find the target node by value
        Map parent = new HashMap<>();
        TreeNode targetNode = buildParents(root, parent, targetVal);
        if (targetNode == null) return Collections.emptyList(); // safety, although target guaranteed present

        // 2) BFS from target node
        List result = new ArrayList<>();
        if (k == 0) {
            result.add(targetNode.val);
            return result;
        }

        Queue q = new ArrayDeque<>();
        Set visited = new HashSet<>();
        q.offer(targetNode);
        visited.add(targetNode);
        int dist = 0;

        while (!q.isEmpty()) {
            int size = q.size();
            if (dist == k) {
                // Collect all nodes currently in queue (they are at distance k)
                for (TreeNode node : q) result.add(node.val);
                return result;
            }
            // Expand one BFS level
            for (int i = 0; i < size; i++) {
                TreeNode node = q.poll();
                // neighbors: left, right, parent
                if (node.left != null && visited.add(node.left)) q.offer(node.left);
                if (node.right != null && visited.add(node.right)) q.offer(node.right);
                TreeNode p = parent.get(node);
                if (p != null && visited.add(p)) q.offer(p);
            }
            dist++;
        }

        // If we exit loop without collecting, no nodes at distance k
        return result;
    }

    // Helper: populate parent map and return the node with value == targetVal (or null)
    private TreeNode buildParents(TreeNode root, Map parent, int targetVal) {
        TreeNode targetNode = null;
        Deque stack = new ArrayDeque<>();
        stack.push(root);
        while (!stack.isEmpty()) {
            TreeNode node = stack.pop();
            if (node.val == targetVal) targetNode = node;
            if (node.left != null) {
                parent.put(node.left, node);
                stack.push(node.left);
            }
            if (node.right != null) {
                parent.put(node.right, node);
                stack.push(node.right);
            }
        }
        return targetNode;
    }

    // --- Example quick tests ---
    public static void main(String[] args) {
        // Build example tree: [3,5,1,6,2,0,8,null,null,7,4]
        TreeNode root = new TreeNode(3);
        root.left = new TreeNode(5);
        root.right = new TreeNode(1);
        root.left.left = new TreeNode(6);
        root.left.right = new TreeNode(2);
        root.left.right.left = new TreeNode(7);
        root.left.right.right = new TreeNode(4);
        root.right.left = new TreeNode(0);
        root.right.right = new TreeNode(8);

        NodesDistanceK solver = new NodesDistanceK();
        System.out.println(solver.distanceK(root, 5, 2)); // expected [7,4,1] (order may vary)
        System.out.println(solver.distanceK(root, 1, 3)); // expected [7,4]?
        // Example 2:
        TreeNode single = new TreeNode(1);
        System.out.println(solver.distanceK(single, 1, 3)); // expected []
    }
}

Tests & edge-cases to mention

  • k == 0 ‚Üí return [target].
  • target is leaf / root: BFS handles both (parents allow upward movement).
  • k larger than tree height ‚Üí [].
  • Single-node tree with k > 0 ‚Üí [].
  • Trees where many nodes are at distance k ‚Äî BFS returns all.

---

Plan (what to say)

  1. Do one traversal to record each node‚Äôs parent (so we can move up the tree).
  2. Run a BFS starting from the target node and expand to its left, right, and parent neighbors.
  3. Stop BFS once we have expanded k levels; all nodes in the queue then are distance k.
This is O(n) time and O(n) space.

---

Why this works

Treat the tree as an undirected graph by adding parent pointers. BFS from the target explores nodes by increasing distance; after k levels the nodes in the frontier are exactly those at distance k.

---

Complexity

  • Time: O(n) (one pass to build parents + BFS up to all nodes).
  • Space: O(n) for parent map + visited set + queue.

Java implementation (BFS + parent map)

/**
 * Definition for a binary tree node.
 * public class TreeNode {
 *     int val;
 *     TreeNode left;
 *     TreeNode right;
 *     TreeNode(int x) { val = x; }
 * }
 */

import java.util.*;

class Solution {
    public List distanceK(TreeNode root, TreeNode target, int k) {
        List result = new ArrayList<>();
        if (root == null) return result;

        // 1) Build parent pointers with a DFS/BFS
        Map parent = new HashMap<>();
        buildParents(root, null, parent);

        // 2) BFS from target
        Queue q = new ArrayDeque<>();
        Set visited = new HashSet<>();
        q.offer(target);
        visited.add(target);

        int dist = 0;
        while (!q.isEmpty()) {
            if (dist == k) {
                // all nodes currently in queue are at distance k
                for (TreeNode node : q) result.add(node.val);
                return result;
            }
            int sz = q.size();
            for (int i = 0; i < sz; i++) {
                TreeNode cur = q.poll();
                // neighbors: left, right, parent
                if (cur.left != null && visited.add(cur.left)) q.offer(cur.left);
                if (cur.right != null && visited.add(cur.right)) q.offer(cur.right);
                TreeNode p = parent.get(cur);
                if (p != null && visited.add(p)) q.offer(p);
            }
            dist++;
        }

        // if we exhaust the graph without reaching distance k
        return result;
    }

    private void buildParents(TreeNode node, TreeNode par, Map parent) {
        if (node == null) return;
        parent.put(node, par);
        buildParents(node.left, node, parent);
        buildParents(node.right, node, parent);
    }
}
