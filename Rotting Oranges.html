Rotting Oranges

Medium
BFSGraphArrayStringTwo Pointers

‚úÖ Problem restatement

We are given a 2D grid where:

  • 0 = empty cell
  • 1 = fresh orange
  • 2 = rotten orange

Each minute, any fresh orange that is 4-directionally adjacent to a rotten orange becomes rotten. We must return the minimum number of minutes until no fresh oranges remain. If it‚Äôs impossible to rot all oranges, return -1.

---

üì• Input

grid = [
  [2,1,1],
  [1,1,0],
  [0,1,1]
]

üì§ Output

4

---

üìé Constraints

  • 1 <= m, n <= 10
  • grid[i][j] ‚àà {0,1,2}

---

üìå Edge Cases to Mention

  • Grid with no fresh oranges ‚Üí return 0 immediately.
  • Fresh oranges isolated by empty cells ‚Üí return -1.
  • All oranges already rotten ‚Üí return 0.
  • Only one rotten orange, surrounded ‚Üí only nearby cells rot.

---

‚úÖ Quick script to say to the interviewer

> ‚ÄúI‚Äôll use BFS from all initially rotten oranges to simulate the spread level by level (minute by minute). > At each level of BFS, I'll rot any adjacent fresh oranges and track time. > If, after BFS, any fresh orange is still left ‚Üí return -1. Otherwise, return total minutes taken.‚Äù

---

üß† Approach: BFS from all rotten oranges

#### Why BFS works:

  • BFS ensures we rot all fresh oranges in the minimum time (by expanding layer by layer).
  • Multi-source BFS handles simultaneous rotting from multiple rotten oranges.

#### Data structures:

  • Queue to store (i, j) of all rotten oranges.
  • Counter for fresh oranges.
  • Direction array to move in 4 directions.

#### Steps:

  1. Initialize queue with all 2s (rotten), and count all 1s (fresh).
  2. While queue is not empty and fresh > 0:
  • For each orange in the current level (1 minute):
  • Try 4 directions. If neighbor is fresh:
  • Rot it (mark as 2), decrement fresh, and add to queue.
  • Increment minutes.
  1. After BFS:
  • If fresh == 0 ‚Üí return minutes.
  • Else ‚Üí return -1.

---

‚è±Ô∏è Complexity

  • Time: O(m * n) ‚Äî visit each cell at most once.
  • Space: O(m * n) ‚Äî queue could store up to all cells.

‚úÖ Java implementation (clean & commented)

import java.util.*;

public class RottingOranges {
    public int orangesRotting(int[][] grid) {
        if (grid == null || grid.length == 0) return -1;

        int m = grid.length, n = grid[0].length;
        Queue queue = new LinkedList<>();
        int fresh = 0;

        // Step 1: Add all initial rotten oranges to the queue, and count fresh ones
        for (int i = 0; i < m; i++) {
            for (int j = 0; j < n; j++) {
                if (grid[i][j] == 2) {
                    queue.offer(new int[]{i, j});
                } else if (grid[i][j] == 1) {
                    fresh++;
                }
            }
        }

        if (fresh == 0) return 0; // No fresh oranges to begin with

        int minutes = 0;
        int[][] directions = {{1,0}, {-1,0}, {0,1}, {0,-1}};

        // Step 2: BFS
        while (!queue.isEmpty()) {
            int size = queue.size();
            boolean rottedThisMinute = false;

            for (int i = 0; i < size; i++) {
                int[] pos = queue.poll();
                for (int[] dir : directions) {
                    int r = pos[0] + dir[0], c = pos[1] + dir[1];
                    if (r >= 0 && r < m && c >= 0 && c < n && grid[r][c] == 1) {
                        grid[r][c] = 2;
                        fresh--;
                        queue.offer(new int[]{r, c});
                        rottedThisMinute = true;
                    }
                }
            }

            if (rottedThisMinute) minutes++;
        }

        return fresh == 0 ? minutes : -1;
    }

    // Quick tests
    public static void main(String[] args) {
        RottingOranges ro = new RottingOranges();

        int[][] grid1 = {
            {2,1,1},
            {1,1,0},
            {0,1,1}
        };
        System.out.println(ro.orangesRotting(grid1)); // Output: 4

        int[][] grid2 = {
            {2,1,1},
            {0,1,1},
            {1,0,1}
        };
        System.out.println(ro.orangesRotting(grid2)); // Output: -1

        int[][] grid3 = {
            {0,2}
        };
        System.out.println(ro.orangesRotting(grid3)); // Output: 0
    }
}

‚öôÔ∏è Optimizations / Alternatives

  • Can optimize space using in-place grid updates instead of a visited matrix.
  • Can use level delimiter (null in queue) instead of level size, but level-size loop is cleaner.
  • For very large grids: could use deque for performance.
