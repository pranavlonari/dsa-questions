Plates Between Candles

There is a long table with a line of plates and candles arranged on top of it. You are given a 0-indexed string s consisting of characters '_' and '|' only, where a '_' represents a plate and a '|' represents a candle.

You are also given a 0-indexed 2D integer array queries where queries[i] = [lefti, righti] denotes the substring s[lefti...righti] (inclusive). For each query, you need to find the number of plates between candles that are in the substring. A plate is considered between candles if there is at least one candle to its left and at least one candle to its right in the substring.

For example, s = "|||||_", and a query [3, 8] denotes the substring "_||\\|". The number of plates between candles in this substring is 2, as each of the two plates has at least one candle in the substring to its left and right. Return an integer array answer where answer[i] is the answer to the ith query.

Example 1:

ex-1 Input: s = "||\\\*|", queries = [[2,5],[5,9]] Output: [2,3] Explanation:

  • queries[0] has two plates between candles.
  • queries[1] has three plates between candles.
Example 2:

ex-2 Input: s = "\||\||||\*", queries = [[1,17],[4,5],[14,17],[5,11],[15,16]] Output: [9,0,0,0,0] Explanation:

  • queries[0] has nine plates between candles.
  • The other queries have zero plates between candles.

Constraints:

3 <= s.length <= 10^5 s consists of '\*' and '|' characters. 1 <= queries.length <= 10^5 queries[i].length == 2 0 <= lefti <= righti < s.length

What to say (one-liner)

‚ÄúPrecompute a prefix sum of plates (*) and for every index the nearest candle | to the left and to the right. For a query [l,r] the first candle inside or after l is a = nextCandle[l] and the last candle inside or before r is b = prevCandle[r]. If a and b exist and a < b then plates between them are prefix[b] - prefix[a], otherwise zero.‚Äù

---

Why it works

prefix[i] counts plates up to index i. If a and b are candle indices with a < b, then prefix[b] - prefix[a] counts plates strictly between a and b. Using nearest-right-from-l and nearest-left-from-r ensures we only count plates that lie inside the query and between two candles.

---

Java code

class Solution {
    public int[] platesBetweenCandles(String s, int[][] queries) {
        int n = s.length();
        int[] prefix = new int[n];        // prefix[i] = number of '*' in s[0..i]
        int[] prevCandle = new int[n];   // prevCandle[i] = index of last '|' at or before i (or -1)
        int[] nextCandle = new int[n];   // nextCandle[i] = index of first '|' at or after i (or -1)

        // build prefix
        prefix[0] = s.charAt(0) == '*' ? 1 : 0;
        for (int i = 1; i < n; i++) {
            prefix[i] = prefix[i - 1] + (s.charAt(i) == '*' ? 1 : 0);
        }

        // build prevCandle (nearest candle to left including itself)
        int last = -1;
        for (int i = 0; i < n; i++) {
            if (s.charAt(i) == '|') last = i;
            prevCandle[i] = last;
        }

        // build nextCandle (nearest candle to right including itself)
        last = -1;
        for (int i = n - 1; i >= 0; i--) {
            if (s.charAt(i) == '|') last = i;
            nextCandle[i] = last;
        }

        int m = queries.length;
        int[] ans = new int[m];

        for (int i = 0; i < m; i++) {
            int L = queries[i][0], R = queries[i][1];
            int a = nextCandle[L];  // first candle at or after L
            int b = prevCandle[R];  // last candle at or before R

            if (a != -1 && b != -1 && a < b) {
                ans[i] = prefix[b] - prefix[a];
            } else {
                ans[i] = 0;
            }
        }

        return ans;
    }
}

---

Complexity

  • Preprocessing: O(n) time, O(n) space.
  • Each query: O(1) time.
  • Total: O(n + q) time and O(n) extra space (where n = s.length(), q = queries.length).
