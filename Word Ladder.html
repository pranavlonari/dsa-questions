Word Ladder

A transformation sequence from word beginWord to word endWord using a dictionary wordList is a sequence of words beginWord -> s1 -> s2 -> ... -> sk such that:

Every adjacent pair of words differs by a single letter. Every si for 1 <= i <= k is in wordList. Note that beginWord does not need to be in wordList. sk == endWord Given two words, beginWord and endWord, and a dictionary wordList, return the number of words in the shortest transformation sequence from beginWord to endWord, or 0 if no such sequence exists.

Example 1:

Input: beginWord = "hit", endWord = "cog", wordList = ["hot","dot","dog","lot","log","cog"] Output: 5

Explanation: One shortest transformation sequence is "hit" -> "hot" -> "dot" -> "dog" -> cog", which is 5 words long.

Example 2:

Input: beginWord = "hit", endWord = "cog", wordList = ["hot","dot","dog","lot","log"] Output: 0 Explanation: The endWord "cog" is not in wordList, therefore there is no valid transformation sequence.

Constraints:

1 <= beginWord.length <= 10 endWord.length == beginWord.length 1 <= wordList.length <= 5000 wordList[i].length == beginWord.length beginWord, endWord, and wordList[i] consist of lowercase English letters. beginWord != endWord All the words in wordList are unique.

class Solution { public int ladderLength(String beginWord, String endWord, List wordList) { } }

Idea (brief)

Treat words as nodes in a graph; edges connect words that differ by one letter. Use BFS from beginWord to find the shortest path to endWord. To generate neighbors fast, precompute intermediate patterns (replace one letter with *) mapping to words, then expand BFS using these patterns.

Correctness

BFS finds the shortest number of steps in an unweighted graph. Each edge corresponds to a single-letter change, so BFS distance + 1 gives the number of words in the sequence.

Complexity

Building patterns: O(L N) where N = wordList.size() and L = word length. BFS: each word/pattern processed once, total O(L N). Space: O(L N) for pattern map and visited sets.

Java implementation

import java.util.*;

class Solution {
    public int ladderLength(String beginWord, String endWord, List wordList) {
        Set words = new HashSet<>(wordList);
        if (!words.contains(endWord)) return 0;

        int L = beginWord.length();
        // pattern -> list of words (e.g., h*t -> [hot, hit])
        Map> adj = new HashMap<>();
        for (String w : words) {
            for (int i = 0; i < L; i++) {
                String p = w.substring(0, i) + '*' + w.substring(i + 1);
                adj.computeIfAbsent(p, k -> new ArrayList<>()).add(w);
            }
        }
        // include beginWord patterns in case it's not in wordList
        for (int i = 0; i < L; i++) {
            String p = beginWord.substring(0, i) + '*' + beginWord.substring(i + 1);
            adj.computeIfAbsent(p, k -> new ArrayList<>());
        }

        Queue> q = new ArrayDeque<>();
        q.offer(new Pair<>(beginWord, 1));
        Set visited = new HashSet<>();
        visited.add(beginWord);

        while (!q.isEmpty()) {
            Pair cur = q.poll();
            String word = cur.getKey();
            int depth = cur.getValue();
            for (int i = 0; i < L; i++) {
                String pattern = word.substring(0, i) + '*' + word.substring(i + 1);
                List neighbors = adj.getOrDefault(pattern, Collections.emptyList());
                for (String nei : neighbors) {
                    if (nei.equals(endWord)) return depth + 1;
                    if (!visited.contains(nei)) {
                        visited.add(nei);
                        q.offer(new Pair<>(nei, depth + 1));
                    }
                }
                // optional: clear list to avoid reprocessing patterns
                adj.put(pattern, Collections.emptyList());
            }
        }
        return 0;
    }

    // Simple Pair class for compatibility
    private static class Pair {
        private final K key; private final V val;
        Pair(K k, V v){ key=k; val=v; }
        K getKey(){ return key; } V getValue(){ return val; }
    }
}

This returns the length of the shortest transformation sequence or 0 if none exists.
